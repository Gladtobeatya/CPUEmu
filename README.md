# CPUEmu

## CPU Emulator

### Registers
R0 to R9

### Instructions
- SET REG REG/IMM → set
- ADD REG REG/IMM → add
- SUB REG REG/IMM → subtract
- JMP IMM → jump
- PRINT REG/IMM → print
- MUL REG REG/IMM → multiply
- DIV REG REG/IMM → divide
- INC REG → increment
- DEC REG → decrement
- CMP REG REG/IMM → compare
- JZ IMM → jump if flag TRUE
- JNZ IMM → jump if flag FALSE
- MOV REG REG/IMM → copy
- HALT → stop CPU

### Usage
Edit `input.txt` to simulate
